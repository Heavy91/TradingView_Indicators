## Indicators:

- **RSI MultiTimeframe:**
  Plots 3 RSI (Weekly, Daily, 4h) at the same time, regardless of the Chart Timeframe.
  Highlights in green (or red) if all RSI are oversold (or overbought).
  Can trigger custom oversold and overbought alerts.
